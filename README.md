# m-aiyouxi-pages

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库中的页面彼此独立，以静态 HTML 文件的形式存放，可直接通过 GitHub Pages 或其他静态托管服务访问。

本仓库不针对任何单一域名或网站，也不包含特定应用或服务的推广内容。页面内容可能涵盖工具、展示页、存档页、个人项目等，具体用途以各页面自身说明为准。

## 目录说明

```
m-aiyouxi-pages/
├── index.html          # 仓库主页（如有）
├── page1/              # 示例页面目录
│   └── index.html
├── page2/
│   ├── index.html
│   └── assets/
└── ...
```

- 每个独立页面通常存放在以页面名命名的子目录中，内含 `index.html` 及相关资源文件（CSS、JavaScript、图片等）。
- 根目录下的 `index.html` 可作为目录页或导航页（可选）。
- 不强制要求统一结构，各页面可按需组织。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务或数据库。
- 页面可能引用外部资源（如 CDN 库、字体、图片），请确保其来源可靠且符合使用许可。
- 部分页面可能为历史存档或实验性内容，不代表最终版本或持续维护。

## 使用方式

1. 克隆或下载本仓库：
   ```bash
   git clone https://github.com/your-username/m-aiyouxi-pages.git
   ```
2. 直接在浏览器中打开某个页面的 `index.html` 文件即可预览。
3. 如需通过 GitHub Pages 发布，请在仓库设置中启用 Pages 功能，并选择 `main` 分支（或 `docs/` 目录）作为源。
